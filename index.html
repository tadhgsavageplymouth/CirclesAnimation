The Social Bubble

Animation 1: Growth Based on View Count and Length

This animation visualises YouTube videos as bubbles growing over time based on view count and video length.

Animation 2: Growth Based on View Count and Recency

This animation shows bubbles growing faster for newer videos, representing growth inversely proportional to days since publication.

Animation 3: Interacting Circles and Emergence

Two circles grow and move towards each other, and upon intersecting, create a new floating bubble.

Animation 4: View Growth Over Time

This animation displays bubbles growing over time to represent the change in view count for each video.

Animation 5: Hashtag Popularity vs View Count

This animation visualizes the popularity of hashtags by relating them to the total view count of videos using each hashtag.
